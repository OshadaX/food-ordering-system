# Food Ordering System — SE2030

## Stack
- Frontend: React 18 + React Router v6 + Axios + Bootstrap 5
- Backend: Java Servlets + Service + Repository + JDBC
- Database: MySQL 8
- Server: Apache Tomcat 10
- Build: Vite (frontend) + Maven (backend)

## Run locally
### Frontend
cd frontend && npm install && npm run dev

### Backend
cd backend && mvn clean install
Deploy war to Tomcat — runs on localhost:8080
